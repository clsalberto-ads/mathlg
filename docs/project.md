# Projeto de linguagem de programação

## Escopo

Nesse projeto estaremos criando uma linguagem de programação especializada em operações matematicas usaremos o python como base.
A linguagem será baseada em uma CLI (Command Line Interface).
A linguagem será baseada em linguagem natural. E devera ser capaz de interpretar comandos em linguagem natural.
Deve conter todos os recursos necessários para realizar operações matematicas em linguagem natural. E ter todas as caracteristicas presentes em uma linguagem de programação completa.
